Mission Statement. Group Project.

PlayFindr is a web service that provides customers with a friendly helpful website to find playgrounds in Ireland. As part of our 3rd year Group Project module we aimed to develop a website that meets the criteria listed in the brief. Our playground finder service allows the user to search for playgrounds in a specific county, get directions to a playground from where they are currently located, view, add or edit playgrounds to the web service, view a 5-day weather forecast for the location and add reviews of playgrounds visited.
